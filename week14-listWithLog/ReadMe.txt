目标：
0.练习用输出到日志文件的方法进行调试。
1.检验上一期作业中 new 和 delete 成对调用。

如何编程：
1.编写调试日志类 DebugLog，可将给定字符串加上时间戳输出到日志文件中。
2.重载 new 和 delete 运算符，每次调用时均产生调试日志。

结果：
一次运行的结果如下：
new called	2015/6/9 12:47:42.199
new called	2015/6/9 12:47:42.202
new called	2015/6/9 12:47:42.205
delete called	2015/6/9 12:47:42.460
delete called	2015/6/9 12:47:42.464
delete called	2015/6/9 12:47:42.469
可见 new 和 delete 确实是成对被调用的。
